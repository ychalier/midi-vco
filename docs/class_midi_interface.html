Detailed Description

Provides handlers for the MIDI library callbacks. It currently supports note-on, note-off, pitch-bend, control-change, after-touch-poly and after-touch-channel.

◆ handle_pitch_bend()

void MidiInterface::handle_pitch_bend ( int  bend)

Callback for the MIDI pitch-bend message. Also updates the channel A of the MidiInteface DAC. The pitch bend value (in the -8192, 8191 range) is mapped to 0, 5000 (corresponding to a 0V-5V ouptut range).

Parameters
bendSigned 14-bit encoding of the current position of the pitch-bend control.
